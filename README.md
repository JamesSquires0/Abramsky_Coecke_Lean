# Lean-CQM: Categorical Quantum Mechanics in Lean 4

This repository contains a formalization of Categorical Quantum Mechanics (CQM) using Lean4.

# Current milestones:
. Definitions of name, coname, and transpose.

# Project Overview

This project provides for Lean formalism for dagger closed compact categories with biproducts to model quantum processes (Abramsky & Coecke, 2004).

# References

S. Abramsky and B. Coecke, "A categorical semantics of quantum protocols," Proceedings of the 19th Annual IEEE Symposium on Logic in Computer Science, 2004., Turku, Finland, 2004, pp. 415-425, doi: 10.1109/LICS.2004.1319636.
